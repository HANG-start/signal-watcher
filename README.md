# Signal Watcher

Real-time Twitter/X monitoring system with automation workflows and notification support.

## Features

- Twitter/X monitoring
- KOL tracking
- Real-time alerts
- Telegram notifications
- Docker deployment

## Tech Stack

- Python
- Docker
- VPS
- Telegram Bot

## Quick Start

```bash
git clone https://github.com/HANG-start/signal-watcher.git

cd signal-watcher

cp .env.example .env

docker compose up -d
```
